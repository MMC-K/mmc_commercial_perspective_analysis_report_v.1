# \[기술문서\] 상용 서비스 관점의 복합대화 기술 성능 분석서 V.1

##	개요
본 보고서는 detour 데이터에서 visualized_output_common 폴더의 html 파일 일부를 검수한 내용을 기반으로 대화형 AI 모델의 성능을 심층 분석한 결과를 담고 있다. 분석 결과, 모델은 지정된 요구사항을 완벽하게 충족하는 성공적인 대화를 생성할 수 있는 잠재력을 보여주었으나, 여러 영역에 걸쳐 일관성 부족 및 명확한 품질 저하 패턴이 식별되었다.
가장 두드러지는 강점은 일부 대화에서 캐릭터, 주제, 이미지를 완벽하게 통합하여 자연스러운 흐름을 만들어내는 능력이라고 생각한다.
반면, 캐릭터의 핵심 설정(직업, 가족 관계, 성별 등)을 대화에 일관되게 반영하지 못하는 문제가 반복적으로 보였다.
또한, 대화의 논리적 흐름이 끊기거나, 생성된 질의응답(QA)이 지문과 무관하게 풀리는 등 전반적인 품질 관리에 개선 포인트가 있었다.
예를들어 동향으로는 대화의 턴 수가 길어질수록 오류 발생 가능성이 높아졌으며, 텍스트와 이미지를 연계하는 문제 생성에서 논리적 비약이 발견되었다.
